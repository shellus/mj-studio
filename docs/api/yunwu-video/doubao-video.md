# 豆包视频生成 API

> 来源：云雾 API
>
> 官方文档：https://www.volcengine.com/docs/82379/1520757
>
> 集成状态：❌ 未集成

## 接口列表

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 文生视频 | 火山引擎格式 | POST |
| 图生视频-首帧 | 火山引擎格式 | POST |
| 首尾帧视频 | 火山引擎格式 | POST |
| 参考图视频 | 火山引擎格式 | POST |
| 查询单个任务 | 火山引擎格式 | GET |
| 查询任务列表 | 火山引擎格式 | GET |

## 模型列表

- `seedance-lite`：轻量版本
- 支持首帧、尾帧、参考图等多种模式

## 特性

- 支持文生视频
- 支持图生视频（首帧）
- 支持首尾帧视频
- 支持参考图视频
- 支持 base64 编码图片
