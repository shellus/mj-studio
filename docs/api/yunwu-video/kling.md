# 可灵（Kling）视频生成 API

> 来源：云雾 API
>
> 官方文档：快手可灵平台
>
> 集成状态：❌ 未集成

## 接口列表

### 文生视频

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 文生视频 | 可灵平台格式 | POST |
| 查询任务 | 可灵平台格式 | GET |

### 图生视频

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 图生视频 | 可灵平台格式 | POST |
| 查询任务 | 可灵平台格式 | GET |

### Omni-Video

| 操作 | 端点 | 方法 |
|-----|------|-----|
| Omni-Video | 可灵平台格式 | POST |
| 查询任务 | 可灵平台格式 | GET |

### 多图参考生视频

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 多图参考生视频 | 可灵平台格式 | POST |
| 查询任务 | 可灵平台格式 | GET |

### 多模态视频编辑

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 初始化待编辑视频 | 可灵平台格式 | POST |
| 增加视频选区 | 可灵平台格式 | POST |
| 删减视频选区 | 可灵平台格式 | POST |

## 特性

- 支持文生视频
- 支持图生视频
- 支持 Omni-Video（多模态视频）
- 支持多图参考生视频
- 支持多模态视频编辑
- 支持 Callback 协议
