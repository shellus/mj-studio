# 海螺（MiniMax）视频生成 API

> 来源：云雾 API
>
> 官方文档：https://www.minimax.io/platform/document/video_generation
>
> 集成状态：❌ 未集成

## 接口列表

| 操作 | 端点 | 方法 |
|-----|------|-----|
| 提交视频生成任务 | `/v1/video_generation` | POST |
| 视频任务状态查询 | `/v1/query/video_generation` | GET |
| 图生视频 | `/v1/video_generation` | POST |
| 首尾帧视频 | `/v1/video_generation` | POST |

## 特性

- 支持文生视频
- 支持图生视频
- 支持首尾帧视频
